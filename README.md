# 🔐 Web Simulasi Algoritma Kriptografi

> Aplikasi web interaktif untuk mensimulasikan algoritma kriptografi simetris **DES**, **S-DES**, **AES**, dan **S-AES** dengan tampilan perhitungan **step-by-step**.

---

## 📋 Deskripsi

Aplikasi ini merupakan simulasi kriptografi yang dirancang untuk membantu mahasiswa dan praktisi memahami alur kerja algoritma enkripsi-dekripsi simetris secara visual. Setiap modul algoritma menampilkan seluruh tahapan perhitungan secara rinci dan bertahap, mulai dari input, pembentukan subkey, hingga hasil akhir.

### Fitur Utama:
- ✅ 4 Algoritma Kriptografi: **DES**, **S-DES**, **AES**, **S-AES**
- ✅ Mode **Enkripsi** dan **Dekripsi**
- ✅ Tampilan **Step-by-Step** untuk setiap tahapan
- ✅ Landing page dengan navigasi ke semua modul
- ✅ Tampilan **responsif** (Desktop & Mobile)
- ✅ Tombol **Submit** dan **Reset** pada setiap modul
- ✅ Hasil akhir ditampilkan dengan format yang jelas
